# Sonar Design Practice

A collection of 10 mini Java projects (skeletons) for interview practice. Each module is a small self-contained demo under `src/main/java/com/example/sonarpractice/<module>`.

Run any Main class with `mvn -q compile exec:java -Dexec.mainClass="com.example.sonarpractice.<module>.Main"` after adding exec plugin, or import into IntelliJ and run main classes directly.

Modules:
- parkinglot
- library
- vendingmachine
- elevator
- urlshortener
- logger
- notification
- payments
- refactor
- codequality

This archive contains minimal, Java-11 compatible skeletons meant for learning and extension.
